# 色彩

## 设计决策

SongStyle 参考色彩使用瓷白、墨色、雾灰、青瓷与有限暖色，强调低饱和关系而不是高刺激。

## 相关设计变量

参见 [Design Tokens / 设计变量](../../design-tokens/) 中的 `color.*` 组。

## 适合使用

用墨色保证正文对比度，用青瓷色建立有限强调，用暖色表达少量温度。

## 不鼓励的模式

不要把高饱和“国潮”配色当作文化捷径，也不要为了柔和而牺牲可访问性对比度。

## 审查问题

色彩是否帮助理解与信任，而不是只在制造风格？
