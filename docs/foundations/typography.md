# 字体

## 设计决策

中文可读性优先。展示字体应克制使用，正文需要可靠的中英文搭配与稳定行高。

中文标题可读性是硬约束：留白不能制造破碎语义、单字悬挂或为了装饰而变窄的标题列。

## 相关设计变量

参见 [Design Tokens / 设计变量](../../design-tokens/) 中的 `typography.*` 组。

## 适合使用

用宋体类展示字体表达少量标题气质，用无衬线字体保证界面与正文阅读。

## 中文标题边界

中文标题需要通过合适行宽、字号与行高保持完整语义。标题可以使用克制的节奏和留白，但不能为了装饰压成难读窄列，不能造成单字悬挂，也不能拆开“安静”“叙事”等语义单元。若标题需要更安静，应先调整层级、空间和语气，而不是压窄文字。

## 不鼓励的模式

不要把书法字体当作宋式感捷径，也不要让中英文基线、字重与密度失衡。
不要为了制造留白，把中文标题压成难读的窄列。

## 审查问题

文字是否在不同语言、屏幕与阅读时长下保持清楚？
