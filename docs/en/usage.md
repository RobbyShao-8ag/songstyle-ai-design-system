# Using SongStyle AI Design System

## Available Modules

- Core design principles and Chinese design guidance
- [Design Tokens / design variables](../../design-tokens/)
- [Shared review checklist](../../checklist/)
- `prompts/` for structured Web design and review prompts
- `skills/` for portable Agent Skills
- A small [Reference UI](../../reference-ui/)

## How To Use The Assets

Use the Design Tokens as a reference expression layer, not as a substitute for design judgment. Use prompts when you need a copyable structured request. Use Agent Skills when an AI agent should follow a repeatable workflow and self-review against the shared model.

Browse the rendered [prompt templates](../../prompts/).

## What v0.1 Does Not Include

SongStyle v0.1 is not a production component library, an online AI generator, a Figma plugin, an MCP server, or a historical reconstruction.

## Run Locally

```bash
npm install
npm run dev
npm run verify
```

Read the [manifesto](../manifesto/) and [core principles](../core-principles/) before applying the visual language.
