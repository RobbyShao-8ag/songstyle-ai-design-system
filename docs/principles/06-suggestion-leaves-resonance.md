# 含蓄留韵 / Suggestion Leaves Resonance

`suggestion-leaves-resonance`

设计不必一次解释全部意义，但不能隐藏必要信息。

## 这意味着什么

含蓄不是故意难懂。它允许品牌气质、情绪和故事通过选择与留白被感受，同时让任务信息保持清楚。

## 为什么重要

当页面把每一层意义都解释完，观看者没有参与理解的空间；当页面隐藏任务信息，含蓄又会变成阻碍。

## 如何转译为现代 Web 设计

- 在生活品牌首页中，用一个准确场景与简短文字暗示生活方式，详细规格放在需要的位置。
- 在数字产品中，首页先表达核心价值，再让用户逐步发现高级能力，而不隐藏试用入口。

## 常见误区

- 用模糊文案代替清楚价值主张。
- 为了留下余韵而隐藏价格、状态、错误或关键操作。
