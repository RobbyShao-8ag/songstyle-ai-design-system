# 平淡见真 / Quietness Reveals Truth

`quietness-reveals-truth`

设计通过真实内容、准确结构与长期可读性建立价值，而不是依赖强刺激。

## 这意味着什么

安静的设计不需要在第一秒证明自己“很设计”。它让真实产品、清楚语言和稳定结构经得起重复使用。

## 为什么重要

视觉技巧能够吸引短暂注意，却不能代替可信信息。对于长期使用的数字产品，可读性与结构准确性比新奇更重要。

## 如何转译为现代 Web 设计

- 用真实材料、规格和使用场景建立生活品牌的可信度。
- 用清楚的功能说明、引用与状态反馈证明数字产品能力。

## 常见误区

- 把平淡误解为缺少观点、缺少品牌性格。
- 用含糊文案和低对比度制造所谓“高级感”。

## 审查问题

**在第一次视觉印象之后，它仍然会显得有价值吗？**
