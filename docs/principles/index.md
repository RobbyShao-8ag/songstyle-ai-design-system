# 宋式设计六项核心原则

这六项原则是 SongStyle 的核心层。它们可以应用于不同品牌和行业，不要求页面看起来历史化，也不要求使用固定配色。

1. [少而有序](less-with-order/)：减少不是目的，关系才是目的。
2. [留白生意](space-creates-meaning/)：空间必须承担功能。
3. [克制有度](restraint-knows-measure/)：表达充分后应当停止。
4. [平淡见真](quietness-reveals-truth/)：通过真实内容与准确结构建立价值。
5. [温润可亲](warmth-makes-technology-approachable/)：让技术拥有可信赖的人性尺度。
6. [含蓄留韵](suggestion-leaves-resonance/)：保留联想空间，但不隐藏必要信息。

使用[宋式审查清单](../checklist/)把原则转化为可重复的设计判断。
