# 留白生意 / Space Creates Meaning

`space-creates-meaning`

留白是用于聚焦、节奏、呼吸与联想的主动设计元素。

## 这意味着什么

空间不是没有内容的剩余区域。它应帮助观看者理解哪些内容属于一起、哪些内容需要停顿、哪些内容值得被首先看见。

## 为什么重要

当界面被均匀填满时，所有信息都在争夺注意力。功能性留白让作品不依赖强刺激，也让观看者有时间完成自己的理解。

## 如何转译为现代 Web 设计

- 在落地页中，用较大的段落间距分隔不同叙事阶段，而不是在每段之间增加装饰图形。
- 在表单中，用空间区分任务分组，同时保持标签、输入框与错误信息彼此接近。

## 常见误区

- 把大面积空白当作高级感，却让用户找不到下一步。
- 在信息密集型产品中强行拉开距离，降低比较效率。
