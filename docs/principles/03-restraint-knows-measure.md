# 克制有度 / Restraint Knows Its Measure

`restraint-knows-measure`

当信息、色彩、装饰与动效完成任务后，设计就应当停止。

## 这意味着什么

克制不是拒绝表达，而是要求每一次新增都证明自己的作用。工具能够继续生成，不代表设计需要继续添加。

## 为什么重要

过量视觉效果会提高理解成本，使品牌气质依赖短暂刺激，并让真正重要的行动失去清晰度。

## 如何转译为现代 Web 设计

- 当主要按钮已经通过位置、文案和色彩清晰表达时，不再叠加发光、渐变和循环动画。
- 当产品截图已经证明功能时，不再增加无关的粒子背景或文化符号。

## 常见误区

- 把克制理解为所有页面都必须低对比、无动效。
- 只删除装饰，却继续堆叠重复卖点和竞争行动。

## 审查问题

**这个新增内容是在改善意义，还是只是在证明还能添加更多？**
