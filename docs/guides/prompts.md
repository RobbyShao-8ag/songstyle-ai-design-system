# SongStyle Prompt 模板

四个 Prompt 模板共享六项原则与同一份审查清单。它们要求 AI 说明假设、信息层级、删减决策、视觉方向、实现约束与审查结果。

使用 Prompt 前，先完成硬约束检查：可用性、可访问性、业务目标和必要信息不能被审美取舍覆盖。生成或审查方案时，先用 [UI/UX 执行模型](../guides/ui-ux-execution-model/) 做场景匹配、页面模式、交互状态和交付前检查，再用近景、中景、远景分配信息距离，并把删除、延后、合并、保留和澄清写入 [决策记录模板](../guides/decision-record-template/)；发现空白崇拜、标题破碎或低对比等问题时，对照 [反模式](../guides/anti-patterns/) 修正。
