# SongStyle 案例

这些案例证明 SongStyle 可以服务生活方式品牌、现代数字产品和电商介绍图片生成。它们不是品牌模板，而是设计判断的可视化证据。对比案例使用同一生成提示词；Base Prompt 只交代任务和内容，不包含排版、布局、字体、风格、留白或色彩方向；SongStyle 版本只多一句调用 SongStyle Skill 或 Prompt 的要求，避免把比较变成两套完全不同的创意方向。
