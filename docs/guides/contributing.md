# 参与贡献

欢迎贡献设计研究、文档、设计变量、Prompt、Agent Skill、参考 UI 与案例。

提交前请阅读仓库中的 [`CONTRIBUTING.md`](https://github.com/RobbyShao-8ag/songstyle-ai-design-system/blob/main/CONTRIBUTING.md) 与 [`ROADMAP.md`](https://github.com/RobbyShao-8ag/songstyle-ai-design-system/blob/main/ROADMAP.md)。贡献者不得把现代转译描述为历史事实，也不得为了视觉风格牺牲可用性、可访问性与业务目标。

## 当前最需要的贡献

- 使用 SongStyle Prompt 或 Agent Skill 后的真实反馈
- 保持文案、图片、功能和目标不变的公平案例对比
- 对原则、必要密度与停止时机的设计讨论
- 参与[首轮用户测试](/songstyle-ai-design-system/research/first-round-user-testing/)

请优先使用 GitHub 提供的 usage feedback、community case-study proposal 与 design discussion issue forms。贡献意见应包含上下文、证据与预期结果。
