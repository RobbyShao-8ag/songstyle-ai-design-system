# 参与贡献

欢迎贡献设计研究、文档、设计变量、Prompt、Agent Skill、参考 UI 与案例。

提交前请阅读仓库中的 [`CONTRIBUTING.md`](https://github.com/RobbyShao-8ag/songstyle-ai-design-system/blob/main/CONTRIBUTING.md)。贡献者不得把现代转译描述为历史事实，也不得为了视觉风格牺牲可用性、可访问性与业务目标。
