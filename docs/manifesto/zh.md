# AI 时代的宋式设计宣言

AI 让设计更快，也更容易过量。它可以持续生成更多模块、更多卖点、更多视觉效果，却很少主动判断表达何时已经充分。

宋式 AI 设计系统不是对宋代历史设计的复原，也不是一套古风装饰。它尝试把我们从宋代美学中感受到的克制、秩序、温润与余韵，转译为现代 Web 设计可以执行和审查的方法。

极少的信息并不自动产生高级感。只有当信息之间的关系准确、空间布局有目的、阅读路径稳定时，少量表达才会显得完整。留白也不是缺少工作，它给观看者时间与空间，让意义在未被完全说明之处继续生长。

SongStyle 希望教会 AI：在表达已经充分之后停止。删除不是目的，秩序才是目的；安静不是模糊，真实才是基础。

无论审美方向如何，可用性、可访问性、真实性与业务目标都不能被牺牲。更空不等于更好，必要的信息密度永远值得保留。
