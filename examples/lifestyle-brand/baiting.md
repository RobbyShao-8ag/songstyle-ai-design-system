## Product brief and business goal

白汀 Baiting 是一个当代家居饮水与桌面器物品牌。落地页需要介绍旗舰玻璃水壶，表达材料品质与日常使用，提供清楚购买路径，并在不使用古风装饰的前提下建立信任。

两个版本现在都是完整的移动端建站方案，而不是单个展示卡片。

## Ordinary AI default proposal

普通 AI 默认方案使用与 SongStyle 改写完全相同的生成提示词、品牌、标题、说明、产品图片、规格信息、行动与业务目标。它仍然专业、可用，但更接近常见默认落地页：首屏、卡片、按钮和信任说明都出现了，只是层级偏平均，页面没有明确判断哪些信息应该先被看见。

## Default diagnosis

内容本身是必要的，但首屏文字、产品图、规格卡片和业务说明之间的主次不够稳定。它不是错误风格，而是缺少 SongStyle 对“器物、证据、停顿”的组织方式。

## UI/UX 执行模型再生成

这次按新的执行模型重新生成两个移动网站方案。普通版与 SongStyle 版使用同一基础生成提示词；Base Prompt 只交代任务和内容，不包含排版、布局、字体、风格、留白或色彩方向；SongStyle 版本只多一句调用 SongStyle Skill 或 Prompt 的要求，所以比较对象不是两套创意风格，而是同一任务是否经过宋代美学项目的执行层处理。

- **Scenario fit / 场景匹配**：生活方式电商产品落地页，核心任务是理解器物、判断材料可信度并进入购买路径。
- **Page pattern / 页面模式**：普通 AI 默认图使用 standard mobile landing + equal cards；SongStyle 在同一模式内把近景器物、中景规格、远景日常气质重新排序。
- **Interaction states / 交互状态**：主要行动需要清楚的 focus、hover、active 状态；不可用或 loading 状态不能削弱购买路径；error/success 反馈必须用文本说明。
- **Pre-delivery checks / 交付前检查**：检查 reduced-motion、mobile touch、breakpoint、图片裁切、CTA 可见性和正文对比度，确认移动端没有横向滚动或标题破碎。

## Controlled generation prompts

### Base Prompt

```text
请为移动端生成一个当代家居饮水器物品牌落地页。
品牌：白汀 Baiting。
定位：日常饮水器物。
标题：让水回到桌面，安静地成为日常。
说明：一只 1.2 L 高硼硅玻璃水壶，薄而稳的壶口，容易清洁的宽口结构，以及适合每日使用的温润手感。
规格：高硼硅玻璃、1.2 L、宽口易清洁。
行动：查看器物与购买信息。
业务目标：让玻璃水壶显得值得信任、容易理解、适合日常购买，而不是制造虚假的奢侈感。
图片：使用同一张玻璃水壶编辑摄影。
```

## SongStyle decisions

SongStyle 版本把“器物、光、桌面、留白”作为页面秩序：近景是水壶本身，中景是规格信息，远景是安静日常。

- **删减**：不删减必要内容，只移除不承担信息功能的视觉强调。
- **层级**：让同一标题、说明、规格和行动形成清楚阅读顺序。
- **空间**：在产品对象周围保留功能性留白，使材质与比例成为焦点。
- **色彩**：使用瓷白、墨色与有限青瓷色，不借用水墨、印章或纹样。
- **图片**：复用同一张产品图片，通过自然裁切与比例建立信任。

## 审查结果

SongStyle 改写没有获得更好的文案、图片或初始 Brief。它只多了最后一句 SongStyle 指令，但通过版面、层级、字体、空间与停止时机，让相同内容更可信、更长久。

## Implementation notes

案例直接使用 SongStyle 设计变量，并保持按钮、文本对比度与语义结构可访问。
