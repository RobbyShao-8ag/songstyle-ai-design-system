# 从普通 AI 默认方案到 SongStyle

## 公平对比契约

本页比较的是同一份 Brief 下的完整移动网站与电商介绍图片：AI 默认方案与 SongStyle 方案。

白汀、清序与电商介绍图片都采用同一生成提示词的受控对比：普通版使用 Base Prompt；Base Prompt 只交代任务和内容，不包含排版、布局、字体、风格、留白或色彩方向；SongStyle 版本复用完全相同的 Base Prompt，只在末尾多一句“请使用 SongStyle Web Designer Skill 或 SongStyle Prompt 模板进行设计。”

白汀与清序的普通 AI 默认方案与 SongStyle 改写都使用相同文案、相同图片、相同功能和相同业务目标。电商介绍图片案例使用同一商品、同一介绍目标和同一必要信息。允许变化的只有版面设计、信息层级、排版、字体与字号、色彩、背景、图片裁切、空间、卡片和装饰方式。

## 电商介绍图片

新增电商介绍图片对比用于验证 SongStyle 不只处理网站布局，也能约束生成图片。普通 AI 默认图和 SongStyle 图使用同一个内容-only Base Prompt；SongStyle 图只多一句项目 Skill/Prompt 指令。这个对比不预设 SongStyle 必胜：如果默认图保留了更多购买判断证据，应如实记录。

## 信息必要性

两个 SongStyle 改写都保留全部必要内容。它们删减的是不承担信息任务的视觉强调，而不是产品类别、购买路径、来源收集、综合、协作或试用路径。

## 层级清晰度

普通默认方案让标题、行动、图片、卡片和说明更平均地出现。SongStyle 方案为每个页面建立一个清楚故事和一个主要行动。

## 功能性留白

白汀让器物成为焦点；清序让研究阶段形成阅读停顿，同时保留理解产品所需的信息密度。

## 适度停止

默认生成容易在卡片、按钮、标注和背景上继续添加表达。SongStyle 只保留能够解释价值的表达。

## 秩序而非符号

两个改写方案都不依赖水墨、印章、纹样或书法字体。文化气质来自比例、节奏、材料语言与克制。

## 可用性、可访问性与目标

两个方案都保留清楚行动、可读文本、足够对比度与业务目标。更空并不是评价标准。
